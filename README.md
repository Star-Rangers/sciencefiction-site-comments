# Star Rangers — Site Comments

This repo has no code — it exists purely to host the [GitHub Discussions](https://docs.github.com/en/discussions) that power reader comments on [*Star Rangers*](https://github.com/dermot-r-cochran/star-rangers), an interactive science-fantasy novel.

Comments on the site are powered by [giscus](https://giscus.app), which maps each page to a Discussion in this repo. Looking for the story itself, or the site's source code? Head to **[dermot-r-cochran/star-rangers](https://github.com/dermot-r-cochran/star-rangers)** or sciencefiction.site.
